## HTTP

### HTTP 各版本特点

#### HTTP 1.0

- 无状态、无连接（1.1 解决）
- 队头阻塞（2.0 解决）

#### HTTP 1.1

- 持久连接（Connection: keep-alive）
- 请求管道化
- 增加缓存处理（cache-control）
- 增加 Host 字段
- 断点传输（range）

#### HTTP 2.0

- 二进制分帧
- 多路复用（一个 TCP 连接可以进行任意数量的 HTTP 请求）
- 头部压缩
- 服务器推送
